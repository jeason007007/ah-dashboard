# 意健险业务分析系统

## 📊 项目简介

本地化保险数据分析工具，支持意健险（意外险和健康险）业务数据的多维度分析。**所有数据仅在浏览器端处理，不会上传到服务器**，确保数据安全。

## ✨ 核心功能

### 1. 数据导入
- 支持 Excel 格式（.xlsx, .xls）
- 双文件上传：承保数据 + 理赔数据
- 增量追加模式
- 数据结构验证 + 业务逻辑校验

### 2. 核心概览
- 签单保费、已赚保费
- 简单赔付率、已赚赔付率
- 覆盖人数统计
- 年月趋势分析

### 3. 医疗成本分析
- 账单费用拆解（社保统筹/商业险/个人自负）
- 获赔率分析
- 可视化费用流向

### 4. 风险画像
- **集团业务线重点关注列表**（赔付率 > 70%）
- **投保单位高风险列表**（赔付率 > 99%）
- **投保单位黄色预警列表**（70%-99%）
- 分方案赔付表现
- 高频疾病统计

### 5. 运营时效
- 报案时效分析
- 结案周期统计
- 3日结案率

## 🔒 隐私保护

- ✅ **纯本地处理**：所有Excel数据在浏览器端解析和计算
- ✅ **无服务器上传**：不发送任何数据到后端服务器
- ✅ **LocalStorage存储**：数据仅存储在浏览器本地
- ✅ **即用即走**：关闭浏览器后数据清除（除非导出）

## 📦 技术栈

- **React 18** - UI框架
- **Vite** - 构建工具
- **Recharts** - 数据可视化
- **XLSX** - Excel解析
- **Framer Motion** - 动画效果
- **Lucide React** - 图标库

## 📝 Excel数据格式要求

### 承保数据表
| 字段名 | 必填 | 说明 |
|--------|------|------|
| 个人保单号 | ✅ | 唯一标识 |
| 保费 | ✅ | 数字格式 |
| 个人生效日 | ✅ | 日期格式 |
| 个人满期日 | ✅ | 日期格式 |
| 投保单位名称 | ✅ | 用于集团分析 |
| 方案名称 | ✅ | 投保方案 |

### 理赔数据表
| 字段名 | 必填 | 说明 |
|--------|------|------|
| 个人保单号 | ✅ | 关联承保数据 |
| 出险日期 | ✅ | 日期格式 |
| 赔款金额 | ✅ | 数字格式 |
| 报案日期 | ⭕ | 运营分析 |
| 结案日期 | ⭕ | 运营分析 |
| 疾病诊断 | ⭕ | 疾病统计 |
| 账单总金额 | ⭕ | 医疗费用分析 |
| 社保统筹给付 | ⭕ | 医疗费用分析 |
| 自费/乙类自付 | ⭕ | 医疗费用分析 |

## 🛠️ 部署说明

### Vercel 部署（推荐）

1. Fork 本项目到您的 GitHub 账号
2. 访问 [Vercel](https://vercel.com)
3. 点击 "Import Project"
4. 选择您的 GitHub 仓库
5. Vercel 自动识别 Vite 项目并配置
6. 点击 "Deploy" 即可

### GitHub Pages 部署

```bash
# 安装适配器
npm install -D gh-pages

# 修改 vite.config.js 添加 base: '/repo-name/'
# 构建项目
npm run build

# 部署到 gh-pages 分支
npm run deploy
```
